# AllDownloader - Ridzz Edition

Universal Downloader dengan teknik scraping dan API integration.

## Fitur

- **YouTube** (Scraping) - MP4 & MP3
- **Instagram** (Scraping) - Reels & Posts (tanpa login)
- **TikTok** (API) - Video & Audio
- **Twitter** (API) - Video
- **Pinterest** (Scraping) - Image & Video
- **Spotify Alternative** (Search) - Cari lagu, download dari YouTube
- **Facebook** (API) - Video HD/SD
- **Google Drive** (Direct Link) - File download
- **GitHub** (Raw URL) - File download
- **MediaFire** (API) - File download
